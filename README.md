# HR-Analytics-Dashboard-Power-BI
This HR Analytics Dashboard was developed in Power BI to analyze employee attrition patterns and workforce demographics. The dashboard enables HR teams and business leaders to monitor key workforce metrics and identify factors contributing to employee turnover.
Business Objectives

• Analyze employee attrition trends
• Identify high-risk employee segments
• Evaluate attrition across departments and job roles
• Monitor workforce KPIs
• Support employee retention strategies

Dashboard Features
• Employee Count Overview
• Attrition Analysis
• Department-Wise Filtering
• Age Group Analysis
• Gender-Based Attrition
• Salary Slab Analysis
• Job Role Analysis
• Years-at-Company Analysis

Tools Used
• Power BI
• Power Query
• DAX
• Data Modeling
• Data Visualization

Key Insights
• Highest attrition occurs in the 26–35 age group.
• Lower salary bands show increased turnover.
• Sales Executive roles experience the highest attrition.
• Employee attrition declines with longer tenure.

Business Value
The dashboard provides HR teams with actionable insights that can help improve employee engagement, reduce turnover, and optimize workforce planning.
